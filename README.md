# Sales EDA

## Project Overview
This project focuses on exploratory data analysis (EDA) of a sales dataset to understand sales performance, product trends, and territory-wise insights using Python.

## Dataset
The dataset contains sales-related information such as product line, quantity ordered, sales amount, territory, and order details.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Data cleaning and preparation
- Product line analysis (sales and quantity)
- Territory-wise sales and quantity analysis
- Groupby-based aggregations
- Visualizations for business insights

## Key Insights
- Identified top-performing product lines
- Analyzed territories contributing the highest sales
- Compared sales and quantity trends across categories

## Conclusion
This analysis helps in understanding business performance and supports data-driven decision-making.
